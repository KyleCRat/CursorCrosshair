# Changelog

- Update version

## [12.0.1-2]

### Changed
- Added throttle when updating width and height
- Added delay when display changes to ensure correct size
- Added buffer to ensure size goes to edge of screen
- Added offset to make the line 1px in size and allow the actual pixel your cursor hovers not be directly under the lines from the cursor. Allows using eyedropper tool correctly in world of warcraft, etc.
